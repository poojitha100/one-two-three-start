# one-two-three-start

## Overview
A minimal Flask web application.

## Stack
- Python 3.12
- Flask 3.1

## Entry point
`app.py` — defines the Flask app and a single route (`/`) that returns "Helllo, Flask!".

## Running
The "Start application" workflow runs `python app.py`, serving on port 5000 (host `0.0.0.0`), with Flask debug mode enabled.
